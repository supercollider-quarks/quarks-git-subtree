PMouseButton UGenPatterns quark library


superclass: Pattern


based on MouseButton


NOTE: mac osx only, lag is not implemented

only osx, and only sc3.4 or sc3.5 in 32bit mode (i.e. something is broken under 64bit sc3.5)


PMouseButton(minval, maxval, lag, length)

minval (default: 0)

maxval (default: 1)

lag not implemented

length number of values produced (default: inf)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \freq, PMouseButton(400, 800).trace).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PMouseButton(0, 0.2).trace).play

a.stop
