PWrap UGenPatterns quark library


superclass: PInRange


based on Wrap


PWrap(pattern, lo, hi)

pattern

lo (default: 0)

hi (default: 1)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \midinote, PWrap(Pseq([0, 2, 4, 5], inf), 2, 4).trace*5+50).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PWrap(PSinOsc(16), 0.5, 1).trace*0.5).play

a.stop
