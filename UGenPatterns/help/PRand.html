PRand UGenPatterns quark library


superclass: Pattern


based on Rand


PRand(lo, hi, length)

lo (default: 0)

hi (default: 1)

length number of values produced (default: inf)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \midinote, PRand(60, 70).trace).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PRand(0, 1).trace).play

a.stop
