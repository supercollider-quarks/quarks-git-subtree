PLFSaw UGenPatterns quark library


superclass: Pattern


based on LFSaw


PLFSaw(freq, iphase, mul, add, length)

freq (default: 440)

iphase fixed (default: 0)

mul (default: 1)

add (default: 0)

length number of values produced (default: inf)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \midinote, PLFSaw(16, 0.5, 10, 60).trace).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PLFSaw(16*PSinOsc(64, 0, 0.1, 1), 0.5, 0.25, 0.25).trace).play

a.stop
