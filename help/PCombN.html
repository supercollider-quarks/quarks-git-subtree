PCombN UGenPatterns quark library


superclass: FilterPattern


based on CombN


PCombN(pattern, maxdelaytime, delaytime, decaytime, mul, add)

pattern

maxdelaytime fixed (default: 2)

delaytime (default: 2)

decaytime differs from CombN. here feedback factor. (default: 0.5)

mul (default: 1)

add (default: 0)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \midinote, PCombN(Pseq([12, 0, 0, 0, 0, 0, 0, 0, 0, 0], inf), 3, 3, 0.5).trace+50).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PCombN(PImpulse(16), 2, 2, 0.25).trace).play

a.stop
