PWhiteNoise UGenPatterns quark library


superclass: Pattern


based on WhiteNoise


PWhiteNoise(mul, add, length)


mul (default: 1)

add (default: 0)

length number of values produced (default: inf)



// example

s.boot;

a= Pbind(\dur, 1/3, \legato, 0.4, \midinote, PWhiteNoise(10, 60).trace).play

a.stop

a= Pbind(\dur, 1/6, \legato, 0.2, \amp, PWhiteNoise(PSinOsc(64, 0, 0.5, 0.5), 0).trace).play

a.stop
